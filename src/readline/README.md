# Readline

#### Events

- Go **LEFT**, **RIGHT**, **UP**, **DOWN**
> With arrow keys
- One **WORD** (in *vim* meaning) Forward, Backward.
> Ctrl+f, Ctrl+b
- Go at the **BEGINING** of the command-line or at the **END**
> Ctrl+a, Ctlr+e
- **VISUAL** mode
> Ctrl+v
- **COPY**, **CUT**, **PAST** or **ABORT**
> Ctrl+y, Ctrl+x, Ctrl+p, ESC
